# AiPaike Web 项目

这是一个基于 Vue 3、TypeScript 和 Vite 的项目模板，帮助你快速开始开发。

## 技术栈

- **Vue 3**：用于构建用户界面。
- **TypeScript**：为JavaScript添加静态类型。
- **Vite**：用于快速构建和开发。
- **Vue Router**：用于管理应用的路由。
- **Ant Design Vue**：用于UI组件库。
- **Axios**：用于HTTP请求。
- **ECharts**：用于数据可视化。
- **Marked**：用于解析Markdown。
- **Vue Toastification**：用于显示通知。

# 安装和使用
## yarn工具
```bash
yarn install
yarn dev
```
## npm工具
```powershell
npm install
npm run dev
```
